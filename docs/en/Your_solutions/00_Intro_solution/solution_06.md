# 📘 Function Analysis – Local Minima and Maxima

Consider the function:f(x) = 3x^2 - 12x + 7


## 🧠 Theory

To find **local maxima or minima** of a function:

### Method (Derivative Approach)

1. Compute the first derivative:f'(x)

2. Find critical points by solving:f'(x) = 0

3. Use the second derivative:
- If `f''(x) > 0` → local **minimum**
- If `f''(x) < 0` → local **maximum**

---

## ✏️ Step-by-Step Solution

### 🔹 Step 1: Compute the first derivative

f(x) = 3x^2 - 12x + 7
f'(x) = 6x - 12

---

### 🔹 Step 2: Find critical points

Set derivative equal to zero:
6x - 12 = 0
6x = 12
x = 2

---

### 🔹 Step 3: Compute the second derivative

f''(x) = 6

---

### 🔹 Step 4: Determine the type of critical point

Since: f''(x) = 6 > 0


👉 The function has a **local minimum** at `x = 2`.

---

### 🔹 Step 5: Find the function value at x = 2

f(2) = 3(2)^2 - 12(2) + 7
f(2) = 3(4) - 24 + 7
f(2) = 12 - 24 + 7
f(2) = -5


---

## ✅ Final Answer

- The function has a **local minimum** at: x = 2

- The minimum value is: f(2) = -5


👉 There is **no local maximum**, because the parabola opens upward (coefficient of `x²` is positive).

---

## 🎯 Summary

- This is a **quadratic function** (parabola).
- Since the leading coefficient (`3`) is positive:
- The graph opens upward.
- It has **only a minimum point (vertex)**.
