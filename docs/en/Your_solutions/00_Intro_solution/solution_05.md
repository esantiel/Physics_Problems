# 📘 Trigonometry – Vector Components

## 🧠 Theory

A vector in 2D can be decomposed into two perpendicular components:

- **Horizontal component (x-axis)**
- **Vertical component (y-axis)**

For a vector with:
- Magnitude: `A`
- Angle with horizontal: `θ`

### Formulas:

- Horizontal component: $A_x$ = A cos(θ)

- Vertical component: $A_y$ = A sin(θ)

---

## 📌 Given

- Magnitude: `A = 15`
- Angle: `θ = 60°`

---

## ✏️ Step-by-Step Solution

### 🔹 Step 1: Horizontal Component

$A_x$ = A cos(θ)
$A_x$ = 15 * cos(60°)

We know: cos(60°) = 0.5

So: $A_x$ = 15 * 0.5 = 7.5

---

### 🔹 Step 2: Vertical Component

$A_y$ = A sin(θ)
$A_y$ = 15 * sin(60°)

We know: sin(60°) = √3 / 2 ≈ 0.866

So: $A_y$ = 15 * 0.866 ≈ 12.99


---

## ✅ Final Answer

- Horizontal component: $A_x$ = 7.5

- Vertical component: $A_y$ ≈ 12.99
